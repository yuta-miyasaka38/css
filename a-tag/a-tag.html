(3) カーソルonでデザイン変更 ~a:hover~

  • css float徹底解説
  • 配下フォルダ内の画像へ飛ぶ
    ①「:hover{ }でマウスオン時の処理」
    ②「:hover img{ } img画像にマウスオンした時の処理」
    ※テキストは緑色、太字にする
    .a2:hover {
        color: green;
        font-size: 2em;
        font-weight: bold;
      }

    ※画像1枚目は緑の枠線をつける
    .a3:hover img {
        border: solid 10px green;
    }

    ※画像2枚目は、半透明にし、拡大
    .a4:hover img {
        opacity: 0.5;
        width: 400px;
        height: 250px;
    }

(4) クリック前後で変化 ~a:visited~

  • css float徹底解説
  • 配下フォルダ内の画像へ飛ぶ
    ※ a:visited 効くのは、テキストの色、枠線の色のみ (colorのみ)
    ※ hoverの後に書く
    ※ imgの枠線の色を変える(*hoverで枠線が表示される仕様になっていること。visitedでは枠線表示できない)
        a:visited img{
            border-color: silver;
        }
    ※１枚目の画像は、訪問後枠線の色が緑→オレンジになる
    ※ opacity, width, border など color以外は効かない
